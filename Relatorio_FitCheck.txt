## 2. Objetivo

O **FitCheck** é uma aplicação web que permite aos utilizadores acompanharem o seu bem-estar físico e mental através de registos diários de hábitos saudáveis, como horas de sono, ingestão de água, atividade física, humor e alimentação. O sistema também permite definir metas, visualizar históricos e obter feedback automático com base nos dados inseridos.

## 3. Tecnologias Utilizadas

- Node.js
- Express.js
- EJS (Templates)
- CSS3 customizado com responsividade
- MySQL (via Sequelize ORM)
- Sessões Express para autenticação
- HTML5 e JavaScript básico

## 4. Estrutura da Aplicação

```
FitCheck/
├── public/                 # Arquivos estáticos (CSS, imagens)
├── views/                  # Páginas EJS renderizadas
├── controllers/            # Lógica de negócio
├── models/                 # Definição de entidades com Sequelize
├── routes/                 # Rotas divididas por funcionalidade
├── middleware/             # Verificação de sessão e tipo de usuário
├── config/                 # Conexão com base de dados
└── server.js               # Ponto de entrada da aplicação
```

## 5. Funcionalidades

### 5.1. Autenticação
- Login e registo de utilizadores com encriptação de senha via bcrypt.
- Sessão armazenada no servidor com `express-session`.

### 5.2. Perfil
- Formulário com idade, altura e peso.
- Apenas utilizadores `PRO` podem editar os dados; utilizadores `comuns` apenas podem criar.

### 5.3. Registo Diário
- Dados: sono, qualidade do sono, ingestão de água, humor, atividade, intensidade e alimentação.
- Utilizadores `comuns` podem inserir até **3 registos**.
- Utilizadores `PRO` podem inserir ilimitadamente.

### 5.4. Histórico
- Apenas `PRO` pode visualizar todos os dados.
- `Comum` pode visualizar os seus próprios 3 registos via consulta por data.

### 5.5. Consulta por Data
- Página visual para consultar um dia específico.

### 5.6. Metas
- Disponível **somente para utilizadores PRO**.
- Permite definir metas de bem-estar: horas de sono, ingestão de água e peso ideal.

### 5.7. Feedback Automático
- Exibido no dashboard, com base nos dados inseridos e metas configuradas.

## 6. Estilo e Design
